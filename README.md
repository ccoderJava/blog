# 聪聪碎碎念

基于 VuePress 2 + vuepress-theme-hope 搭建的个人技术博客。

## 特性

- 全文搜索
- RSS / Atom / JSON Feed
- Giscus 评论
- PWA 离线支持
- Sitemap 自动生成
- SEO 优化（OG / Twitter Card / JSON-LD）
- PhotoSwipe 图片预览
- 明暗主题切换

## 本地开发

```bash
yarn run dev     # 启动开发服务器 http://localhost:8000
yarn run dist    # 构建静态站点到 dist/
yarn run deploy  # 构建并推送到 GitHub Pages
```

## 技术栈

- VuePress 2 + Vite
- vuepress-theme-hope
- Markdown + YAML frontmatter
- 托管于 GitHub Pages
